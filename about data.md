# 🛰️ Satellite Data Sources

## 🌈 Sentinel-2 (ESA Copernicus)

| Feature | Details |
|---|---|
| Dataset | `COPERNICUS/S2_SR` |
| Resolution | 10m |
| Purpose | RGB + NIR + RedEdge |
| Strength | High vegetation sensitivity |

### Extracted Bands

| Band | Meaning |
|---|---|
| B2 | Blue |
| B3 | Green |
| B4 | Red |
| B8 | Near Infrared (NIR) |
| B5 | Red Edge |

---

## 🌡️ Landsat 8 (NASA / USGS)

| Feature | Details |
|---|---|
| Dataset | `LANDSAT/LC08/C02/T1_L2` |
| Resolution | 30m |
| Purpose | Thermal Surface Temperature |
| Strength | Heat & temperature analysis |

Thermal data is resampled to:
```text
10m spatial resolution
```

for seamless integration with Sentinel imagery.

# 🌍 End-to-End Flow Diagram

```text
┌─────────────────────────────┐
│  Define Region of Interest  │
│        (AOI Selection)      │
└──────────────┬──────────────┘
               │
               ▼
┌─────────────────────────────┐
│ Load Sentinel-2 Imagery     │
│ RGB + NIR + RedEdge Bands   │
└──────────────┬──────────────┘
               │
               ▼
┌─────────────────────────────┐
│ Apply Filtering             │
│ • Date Filtering            │
│ • Cloud Filtering           │
│ • Band Selection            │
└──────────────┬──────────────┘
               │
               ▼
┌─────────────────────────────┐
│ Generate Median Composite   │
│ Noise & Cloud Reduction     │
└──────────────┬──────────────┘
               │
               ▼
┌─────────────────────────────┐
│ Load Landsat-8 Thermal Data │
│ Surface Temperature Layer   │
└──────────────┬──────────────┘
               │
               ▼
┌─────────────────────────────┐
│ Convert Thermal to Celsius  │
│ Resample 30m → 10m          │
└──────────────┬──────────────┘
               │
               ▼
┌─────────────────────────────┐
│ Merge All 6 Bands           │

└──────────────┬──────────────┘
               │
               ▼
┌─────────────────────────────┐
│ Generate Final GeoTIFF      │
│ AI-Ready Multispectral Data │
└─────────────────────────────┘
```

# ⚠️ Important Technical Notes

✅ Sentinel-2 does not provide thermal imagery  
✅ Landsat 8 is used for thermal fusion  
✅ Thermal data is resampled to 10m  
✅ Median compositing reduces cloud artifacts  
✅ Larger date ranges improve image availability in cloudy regions
